# Street Control

Aplicacion web para el control interno y social de la OTB-Miraflores.

Routes:

- login
- main
  - usuarios (solo admin)
  - vecinos
  - cobros
  - reuniones
  - asistencia
  - reportes
  - lecturacion de agua
  - egresos

La aplicacion es capaz de:
  - Realizar el cobro del agua
  - Control de asistencia a las reuniones
  - Emitir recibo
  - Control de deudas
